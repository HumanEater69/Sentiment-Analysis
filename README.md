## 📌 What Is This Project?

This is a **complete, end-to-end NLP Sentiment Analysis pipeline** built on the [Amazon Fine Food Reviews dataset](https://www.kaggle.com/datasets/snap/amazon-fine-food-reviews) from Kaggle.

Starting from raw CSV data and ending with a **fully interactive dark-neon analytics dashboard**, this project covers everything a real data analyst does on the job:

```
Raw Data → Cleaning → Sentiment Scoring → Visualization → Business Insights → Dashboard
```

The entire pipeline runs on a **5,000-row sample** and is designed to be lightweight, reproducible, and visually polished.

## ⚙️ The Full Pipeline — Step by Step

The notebook (`analysis.ipynb`) is broken into **5 clean tasks**:

### ✅ Task 1 — Data Loading & Exploration
- Loads the first **5,000 rows** from `Reviews.csv`
- Inspects dataset shape, column names, data types
- Prints star rating distribution and null value counts
- Libraries: `pandas`, `matplotlib`, `seaborn`, `TextBlob`

### 🧹 Task 2 — Data Cleaning
- Retains only the `Text` and `Score` columns
- Drops all rows with **null, empty, or duplicate** review text
- Resets the index for a clean slate
- **Result: 4,984 clean reviews ready for analysis**

### 🔍 Task 3 — Sentiment Analysis (TextBlob)
- Computes a **polarity score** (−1.0 to +1.0) for every review using TextBlob
- Maps scores to sentiment labels:
  - 😊 **Positive** → polarity > 0
  - 😐 **Neutral** → polarity = 0
  - 😠 **Negative** → polarity < 0
- Prints distribution summary with labelled examples from each class

### 📊 Task 4 — Visualization (3 Charts)
| Chart | Type | What It Shows |
|-------|------|---------------|
| `chart1_bar.png` | Bar Chart | Count of Positive / Neutral / Negative reviews |
| `chart2_pie.png` | Pie Chart | Percentage share of each sentiment class |
| `chart3_custom.png` | Heatmap | Star Rating (1–5) vs. TextBlob Sentiment Label |

All charts saved at **150 DPI**, using a clean `seaborn-whitegrid` style with custom colour coding (🟢 green = positive, 🔴 red = negative, ⚫ grey = neutral).

### 💡 Task 5 — Insights & Written Summary
- Computes key statistics: mismatch counts, polarity-by-rating trends, avg review lengths
- Surfaces surprising mismatches with real review examples
- Exports a **one-page PDF report** using matplotlib's PDF backend

## 🚨 Interesting Mismatches Discovered

> TextBlob is rule-based and doesn't understand sarcasm or context — which led to some fascinating outliers:

- ❗ **122 reviews** were rated **5-stars** by the user but classified as **Negative** by TextBlob
  - _Why?_ Sarcastic or ironic writing that TextBlob can't detect
- ❗ **248 reviews** were rated **1-star** but classified as **Positive** by TextBlob
  - _Why?_ Polite or mixed phrasing that sounds positive on the surface

These mismatches highlight a **real-world limitation of lexicon-based NLP** and are a strong argument for upgrading to transformer-based models like BERT or RoBERTa for production use.

## 💼 Business Recommendation

> 🎯 **Prioritize product quality and description accuracy for 3-star products.**

The 3-star segment shows the **highest share of Neutral / uncertain sentiment** — 2.5% of all 3-star reviews (10 out of 393) sit in the Neutral bucket. These are fence-sitters: customers who aren't delighted but aren't fully disappointed either. Improving product consistency and description accuracy in this tier offers the **highest-leverage opportunity** to convert undecided buyers into loyal repeat customers.

## 🚀 How to Run This Project

### 1️⃣ Clone the repository
```bash
git clone https://github.com/your-username/sentiment-analysis-amazon.git
cd sentiment-analysis-amazon
```

### 2️⃣ Install dependencies
```bash
pip install pandas textblob matplotlib seaborn
python -m textblob.download_corpora
```

### 3️⃣ Download the dataset
Get `Reviews.csv` from [Kaggle — Amazon Fine Food Reviews](https://www.kaggle.com/datasets/snap/amazon-fine-food-reviews) and place it in the project root.

### 4️⃣ Run the notebook
```bash
jupyter notebook analysis.ipynb
```
Run all cells top to bottom — charts will be saved to `charts/` and the PDF to `summary.pdf`.

### 5️⃣ Generate the interactive dashboard
```bash
python build_dashboard.py
```
Open `dashboard.html` in your browser. No server needed. ✅
